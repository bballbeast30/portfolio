##### Note: This app is in progress

- An app to connect the community in a neighborhood.
- Users can create a community in which posts can be made, and users can schedule events which can be attended by others in the community
- Flutter SDK used to build the app with data from backend fetched from REST API.
- Features include direct messaging, group messaging, and ability to add advertisement per community.
